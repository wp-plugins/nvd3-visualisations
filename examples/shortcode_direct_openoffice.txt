<h3>Data Input from OpenOffice Calc tables</h3> 

By using "class" option you can generate the chart from the tables of OpenOffice easily too.

To do this:
* You can mark input data values & labels on cells with same background color with OpenOffice Calc. 
* Next, you normally copy & paste whole table into WordPress page/post (in visual mode).
* Finally, you just name your table with own ID tag and call it by shortcode (in html mode).

Benefits: you edit table's values directly and the chart updates automatically to show them out.

Basically, all you need to do is copy&paste table in visual mode, add its own ID tag in html mode, and call its drawing with the right class option. Below is whole story.

<b>Simple data set from OpenOffice Calc</b>

Typically, short table after copy & paste on WP side when data values have background color "#FFFF99" (eq Yellow by OpenOffice palette) ...

<table border="0" frame="VOID" rules="NONE" cellspacing="0"><colgroup><col width="86" /><col width="86" /><col width="86" /></colgroup>
<tbody>
<tr>
<td align="LEFT" width="86" height="17">Cats</td>
<td align="LEFT" width="86">Cows</td>
<td align="LEFT" width="86">Birds</td>
</tr>
<tr>
<td align="RIGHT" bgcolor="#FFFF99" height="17">7</td>
<td align="RIGHT" bgcolor="#FFFF99">2</td>
<td align="RIGHT" bgcolor="#FFFF99">12</td>
</tr>
</tbody>
</table>

You just add ID "mypets' for this table ...

<table id="mypets" border="0" frame="VOID" rules="NONE" cellspacing="0"><colgroup><col width="86" /><col width="86" /><col width="86" /></colgroup>
<tbody>
<tr>
<td align="LEFT" width="86" height="17">Cats</td>
<td align="LEFT" width="86">Cows</td>
<td align="LEFT" width="86">Birds</td>
</tr>
<tr>
<td align="RIGHT" bgcolor="#FFFF99" height="17">7</td>
<td align="RIGHT" bgcolor="#FFFF99">2</td>
<td align="RIGHT" bgcolor="#FFFF99">12</td>
</tr>
</tbody>
</table>

And call it with id + bgcolor for values ...

[loadNVD3]
[jsChart type='discretebar' class='{id:"mypets", bgcolor:"#FFFF99"}']

Not so good since labels are numbers, huh? So here we go...

<b>Values + Labels from OpenOffice Calc Automatically</b>

Generating your labels of chart is as easy too: you just mark the different background color for them on OpenOffice side. Here is the same table after copy & paste (in visual mode!) + added another ID for it ...

<table id="mypets2" border="0" frame="VOID" rules="NONE" cellspacing="0"><colgroup><col width="86" /><col width="86" /><col width="86" /></colgroup>
<tbody>
<tr>
<td align="LEFT" bgcolor="#66CC99" width="86" height="17">Cats</td>
<td align="LEFT" bgcolor="#66CC99" width="86">Cows</td>
<td align="LEFT" bgcolor="#66CC99" width="86">Birds</td>
</tr>
<tr>
<td align="RIGHT" bgcolor="#FFFF99" height="17">7</td>
<td align="RIGHT" bgcolor="#FFFF99">2</td>
<td align="RIGHT" bgcolor="#FFFF99">12</td>
</tr>
</tbody>
</table>

And call it again but adding now "titlecolor" option ...

[loadNVD3]
[jsChart type='discretebar' class='{id:"mypets2", bgcolor:"#FFFF99", titlecolor:"#66CC99"}']

Note: labels of chart may come freely from one row / column of OpenOffice table as long as you just mark their bgcolor properly.

<b>Adding Own Custom Labels - Cell by Cell to Each Value</b>

In order to get own labels into chart you just add IDs also for each cells of table like this (some copy&paste stuff removed for the sake of clarity) ...

<table id="mypets3"><tr>
<td id="Dogs" align="RIGHT" bgcolor="#FFFF99" height="17">7</td>
<td id="Hawks" align="RIGHT" bgcolor="#FFFF99">2</td>
<td id="Owls" align="RIGHT" bgcolor="#FFFF99">12</td>
</tr></table>

And shortcode call stays the same ... 

[loadNVD3]
[jsChart type='discretebar' class='{id:"mypets3", bgcolor:"#FFFF99"}']

<b>
That is all folks - enjoy it with OpenOffice for all your tables !
